<h1 align="center"> Projeto PowerSun</h1>


    <h4 align="center"> 
        :construction:  Projeto em construção  :construction:</h4>
        O projeto foi elaborado para mostrar outras alternativas de consumo de energia de forma sustentável e renovável.
Dentre tantas a escolhida fora a Energia Fotovoltaica : A solar. 
Uma fonte de energia limpa, renovável e sustentável. Visando o compromisso com a preservação ambiental.
Sendo esse objetivo principal, é levar também para as empresas, e para além delas a importância e a força que tem esse sistema fotovoltaico.

<h4>Tecnologias utilizadas</h4>

JAVASCRIPT, CCS, HTML
